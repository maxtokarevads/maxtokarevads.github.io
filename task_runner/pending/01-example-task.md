# Example task: Write a short market overview

## Goal
Write a concise (300–400 words) overview of the current state of programmatic
advertising in 2025–2026. Focus on:
- The shift toward AI-driven bidding
- Privacy changes (cookieless, ATT, etc.)
- What it means for small-to-medium advertisers

## Deliverables
- `market_overview.md` — the article itself, in Markdown
- `SUMMARY.md` — standard task summary (auto-required)

## Constraints
- English only
- No hallucinated statistics — if you cite a number, it must come from a
  reputable source you can name
- Keep it practical, not academic
