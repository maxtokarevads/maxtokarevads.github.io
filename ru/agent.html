Analyze — Campaign Diagnostics
Funnel · creative fatigue · attribution · Learning
/analyze · meta
  • Metrics: CPM, CTR, CPA, ROAS, CVR, Frequency, Hook Rate, Video CR
  • Creative fatigue detection: Frequency >3 + CTR drop + rising CPM
  • Learning Limited: causes and how to resolve (budget / events / structure)
  • Attribution inflation: Meta vs GA4 gap — when not to trust platform ROAS
  • Funnel breakdown: CPM → CTR → CVR → ROAS (where it breaks)
  • 2026 benchmarks: Link CTR ≥0.9%, Hook Rate ≥25%, Frequency ≤5/7d

Copy — Ad Copywriting
3 hook variants · TOFU / MOFU / BOFU
/copy · meta
  • Variant 1 — Hook: Pain / Problem (open with the audience's pain)
  • Variant 2 — Hook: Number / Result (specific stat or outcome)
  • Variant 3 — Hook: Question / Intrigue (a "yes" they want to say)

A/B Test — Experiment Design
Meta Experiments · Holdout · ABO vs CBO
/ab_test · meta
  • Meta Experiments: Creative / Audience / Placement / Budget Strategy
  • Holdout Test: 10–15% sees no ads — measures true incremental lift
  • ABO vs CBO: 14-day test with equal total budget
  • Tests: format, audience (interests vs Advantage+ vs LAL vs broad)
  • Sample size for each branch: ≥50 conversion events per 7 days
